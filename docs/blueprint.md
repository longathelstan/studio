# **App Name**: LinkWave

## Core Features:

- URL Input: Accept a long URL from the user.
- Short Link Creation: Generate a shortened URL from the provided long URL.
- QR Code Generation: Give the user the option to automatically create a QR code for the shortened URL. Use an AI tool to determine whether a QR code would be useful to the user, depending on the specific characters that compose the shortened URL.
- Short Link Display: Display the shortened URL to the user.
- QR Code Display: Display the generated QR code to the user.
- Copy to Clipboard: Provide buttons to copy the short URL or QR code to the clipboard.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB), evokes a sense of trust and clarity.
- Background color: Very light blue (#F0F8FF), provides a clean and airy feel.
- Accent color: Electric blue (#7DF9FF) for interactive elements and highlights, providing a modern and techy feel.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable design.
- Use simple, line-based icons in a light blue tone.
- Use a clean and minimalistic layout to focus on the core functionality.
- Use subtle animations and transitions for user interactions.
